Microsoft og tekniske specialister fra vores samarbejdspartnere, inviterer til en dyb teknisk gennemgang (level 300) af udvalgte funktioner inden for Windows Server 2012 og nyhederne i Service Pack 1 til System Center.
Kom til en deep dive den 6. marts 2013 hos Microsoft i Hellerup.
Formiddags session:
Med over 300 nye features i Windows Server 2012 kan det være en udfordring at komme i gang med den praktiske anvendelse af det nye OS. Vi har derfor udvalgt 4 områder, hvor du hurtigt kan få fordel af de nye funktionaliteter.
Eftermiddags session:
SP1 til System Center indeholder flere væsentlige opdateringer som kan optimere din eksisterende System Center implementering. Vi gennemgår nyhederne i SP1 med fokus på datacenter og client management.
