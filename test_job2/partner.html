I September 2012 lancerede Microsoft Windows Server 2012, men der erstadig mange brugere I SMV, som benytter en ældre version af Windows Server.
Hele 77 % af SMV'er kører i dag en version afWindows Server – det svarer til omkring 22,5 millioner servere i hele verden. Mere end halvdelen (57 %) af disse kører Windows Server 2003, som er et 10 år gammelt operativsystem.
